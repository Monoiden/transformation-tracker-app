# Transformation Tracker App

Diese Version ist eine funktionierende Next.js-Web-App, die Daten zunächst lokal im Browser speichert.
Damit kannst du sie sofort auf Vercel deployen und direkt auf dem iPhone testen.

## Lokal starten
```bash
npm install
npm run dev
```

## Auf Vercel deployen
- Vercel öffnen
- Add New Project
- Repository auswählen
- Framework: Next.js
- Deploy klicken

## Auf dem iPhone installieren
- App-URL in Safari öffnen
- Teilen
- Zum Home-Bildschirm
